This is a responsive react template made with [Create React App](https://github.com/facebook/create-react-app).

## Available Features

#### Responsive for all screen sizes.
#### Comes with Navigation.
#### Includes navigation using react router.

## Installed Libraries

* react bootstrap
* react router
* react router dom
